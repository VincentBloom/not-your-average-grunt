2. Create a Hello World in NodeJs and create a tutorial about it. 15 points.

Part 1. Installing Node.js

Note: Using Ubuntu 16.04.02. Check your local OS to know how to install.
  1. Open Terminal (Cntrl+Alt+t, on desktop).
  2. Run the following commands. Using the apt packet manager to refresh the local package index. Then run the next command to install or refresh the Node.js installation. 
    
sudo apt-get update
sudo apt-get install nodejs
  3. Install the Node Package Manager, which will allow us to manage packages in Node.js for the later question. 
    
sudo apt-get install npm

Part 2. Creating your first Node.js program.

  1. Create a file named Hello.js. To do so, run the following command: 
    
sudo gedit Hello.js
    We will use Gedit (a simple text editor) to create a file. This command will open Gedit and name the file Hello.js. Do not close Terminal or your document will not save unless saved.
  2. In Gedit, type: 
    
console.log('Hello world!');
    This command tells the console to output Hello world!. Such a simple program!
  3. Save the document created by doing the command Ctrl+S. Close the document by running the command Alt+F4.
  4. Run the javascript file you just created by typing in the same console instance: 
    
nodejs Hello
    Note that you do not need the extension to run this command! Important: You must be in the same directory as your Hello.js file for this to work. The console should display in a new line Hello world!.
  5. There you have it, you just installed Node.js, created your first Node.js program and ran it!

3. Install the aws sdk for nodejs and create a tutorial about it. 15 points.

Part 1. Create an empty project with node.

  1. Open terminal, like in previous step.
  2. cd into a desired directory where you wish to create a Node.js project. For example: 
    
cd Documents
  3. Create a new directory in which your project will be held. In this example the project will be named BlankProject. 
    
mkdir BlankProject
  4. cd into your newly created directory. 
    
cd BlankProject
  5. Create a new Node.js project! We will use the Node Package Manager to aid us here. It will create us a package.json file to which we will be able to control our Node.js package installations. 
    
npm init -f
    We use the -f option because we just want a blank default Node.js project. The output of this command is what the package.json looks like!

Part 2. Install AWS SDK in Node.js

Make sure that you are in the correct directory, the root of your newly created Node.js project. Run the command: 


npm install aws-sdk --save-dev
This will download and install the Amazon Web Services SDK for use of your current project. The --save-dev modifiers make sure that the referenced package is saved in package.json to allow free distribution of your project, saving the dependencies for developer use.
